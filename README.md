# Bierkasternerkennung
Python-Anwendung zur Echtzeit-Erkennung des Füllzustands eines Bierkastens mittels Computer Vision. System lokalisiert den Kasten im Bild, unterteilt ihn und erkennt pro Slot, ob eine Flasche vorhanden ist. Anschließend wird über Farbanalyse im HSV-Raum bestimmt, ob die Flasche durch einen Kronkorken als voll/leer klassifiziert wird.
